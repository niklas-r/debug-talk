console.log and friends

console.log(object [, object, ...])

Displays a message in the console. You pass one or more objects to this method, each of which are evaluated and concatenated into a space-delimited string.

console.debug(object [, object, ...])

Same functionality as console.log() but output text is blue.

console.info(object [, object, ...])

Same functionality as console.log() but styled with an info sign.

console.warn(object [, object, ...])

Also same functionality as console.log() but styled with a warning sign.

console.error(object [, object, ...])

Similar to console.log(), console.error() and also includes a stack trace from where the method was called. Doesn't halt code execution as an uncaught exception would.
